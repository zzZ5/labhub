# 门户内容字段映射

本文档记录“门户内容”后台、公开 API 与网站展示位置之间的唯一对应关系。业务内容不得再由前端写死或由演示种子自动生成。

| 后台栏目与字段 | API 字段 | 公开网站位置 | 空值处理 |
| --- | --- | --- | --- |
| 站点首页 / 实验室名称 | `site_name` | 导航品牌、默认横幅标题、页脚名称 | 使用项目品牌兜底 |
| 站点首页 / 归属单位 | `site_subtitle` | 导航副标题、页脚单位 | 使用学院名称兜底 |
| 站点首页 / 横幅副标题 | `hero_subtitle` | 默认横幅及未填写副标题的轮播横幅 | 使用简短研究定位兜底 |
| 站点首页 / 简介正文 | `description` | 首页“课题组简介”正文 | 空值时不显示正文 |
| 站点首页 / 横幅切换间隔 | `banner_interval_seconds` | 首页轮播定时器 | 限制为 3 至 30 秒 |
| 站点首页 / 加入我们说明 | `ContactInfo.content` | 首页“加入我们” | 空值时不显示说明 |
| 站点首页 / 联系邮箱 | `ContactInfo.email` | 首页联系按钮 | 空值时不显示按钮 |
| 页脚设置 / Logo | `logo` | 导航与页脚品牌图 | 使用项目图标兜底 |
| 页脚设置 / 网站图标 | `favicon` | 浏览器图标 | 使用 Logo 或项目图标兜底 |
| 页脚设置 / 默认横幅图 | `hero_image` | 没有有效轮播横幅时的首页横幅 | 使用项目默认横幅 |
| 页脚设置 / 页脚简介 | `footer_text` | 网站页脚左侧 | 使用简短品牌说明兜底 |
| 页脚设置 / 地址 | `address` | 网站页脚右侧 | 使用西校区地址兜底 |
| 页脚设置 / 相关链接 | `external_links` | 网站页脚外链 | 使用学校、学院链接兜底 |
| 首页横幅 | `HomeBanner.*` | 首页轮播图、标题、副标题和可选跳转 | 仅展示启用且有图片的内容 |
| 研究方向 | `ResearchDirection.*` | 首页研究方向、研究方向列表与详情 | 首页最多展示前 6 项 |
| 团队成员 | `Member.*` | 首页成员、成员列表与详情 | 仅展示 `sort_order > 0` 的成员 |
| 新闻活动 | `NewsArticle.*` | 首页新闻、新闻列表与详情 | 仅展示已发布的公开新闻 |
| 论文成果 | `Publication.*` | 首页成果、成果列表与论文详情 | 首页按排序规则选择，列表按年份倒序 |
| 科研项目 | `Project.*` | 首页成果、项目列表与详情 | 首页按排序规则选择，列表按日期倒序 |
| 专利成果 | `Patent.*` | 首页成果、专利列表与详情 | 首页按排序规则选择，列表按日期倒序 |
| 获奖成果 | `Award.*` | 首页成果、获奖列表与详情 | 首页按排序规则选择，列表按日期倒序 |

## 首页成果排序

- 任一成果设置了大于 0 的首页排序后，只展示排序大于 0 的成果，数字越小越靠前。
- 多种成果排序值相同时，按论文、项目、专利、获奖交替展示，并在同类中按日期从新到旧排列。
- 所有成果排序均为 0 时，首页自动展示最新论文；完整成果页仍分别展示论文、项目、专利和获奖。

## 内容分类

新闻分类统一为：学术交流、组内动态、科研进展、成果荣誉、招生招聘、项目动态。研究方向不再维护或展示关键词字段。
